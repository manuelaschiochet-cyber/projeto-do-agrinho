# 🌱 Projeto Agrinho

Este é um projeto desenvolvido para o **Programa Agrinho**, com o objetivo de promover a conscientização sobre temas relacionados ao campo, sustentabilidade, meio ambiente e a conexão entre o rural e o urbano.

## 📌 Sobre o projeto

O projeto busca demonstrar a importância da agricultura, do cuidado com o meio ambiente e do desenvolvimento sustentável, utilizando tecnologias como HTML, CSS e JavaScript (ou outra que você esteja usando).

## 🎯 Objetivo

- Promover a educação ambiental
- Mostrar a importância da agricultura no dia a dia
- Incentivar práticas sustentáveis
- Aproximar o campo da cidade

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- (adicione outras se quiser)
